Angular Building Blocks

Relation between building blocks of an Angular application
  • Angular view is defined by a component and a template:
    • A decorator on a component class adds the metadata, including a pointer to the associated template.
    • Directives and binding markup in a component's template modify views based on program data and logic.
  • The dependency injector provides services to a component, that provide specific functionaliy not directly related to views and information or that lets you define navigation among views, such as the router service.
