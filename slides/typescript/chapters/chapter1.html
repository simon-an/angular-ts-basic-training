What is a JavaScript Engine?

A JavaScript engine is a program or interpreter which executes JavaScript code. A JavaScript engine may be a traditional interpreter, or it may utilize just-in-time compilation to bytecode in some manner. Although there are several uses for a JavaScript engine, it is most commonly used in Web browsers.

JavaScript Engines

  • V8: Google Chrome, NodeJS
  • Chakra: Microsoft Browsers
  • Rhino, Nashorn: Java
  • SpiderMonkey: Netscape, Firefox
  • JavascriptCore: Apple

What is TypeScript?

Docs

  • TypeScript is a syntactic sugar for JavaScript.
  • TypeScript syntax is a superset of ECMAScript 2015 (ES2015) syntax.
  • TypeScript transpiles to JavaScript.

Why should I use TypeScript?

Development with TypeScript is more efficient than development with JavaScript!

  • Compiler recognizes syntax errors.
  • Compiler checks types.
  • d.ts files contain type definition for existing JavaScript libraries.
  • Use ES2016+ in Code, run ES5/ES6 => transpiling + polyfill
TypeScript compiles to ES5 and runs in older Browsers that do not support for example ES6

Compilation

JavaScript is a interpreted language so it has to be executed first before it can be checked for validitiy. So you write the code first and then look if there is an output. If there is no output there will be an error. In that way you need much time to find an error. The TypeScript transpiler has the feature to check for errors. TypeScript compiles the code and will throw a compilation error if it finds a compile error. So errors will be found before executing the script.

Strong Static Typing

JavaScript has no strong types. TypeScript gives you the opportunitiy to use static typing through its TLS (TypeScript Language Service). The TLS will automatically add a type to a variable if there is no type defined explicitly.

Type Definition

TypeScript supports type definition if existing external JavaScript libararies. With these type definitions you can integrate these JavaScript libararies in you TypeScript code.

Object-oriented Programming

TypeScript supports concepts of object-oriented programming like interfaces and inheritance.
